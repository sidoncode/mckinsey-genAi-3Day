# Session 3: Model Evaluation and Comparison

**Time:** 1:30 – 3:15 (1 hour 45 minutes)

## Overview

This session covers how to systematically evaluate and compare LLM models for agentic tasks. Participants will learn to define evaluation metrics, use LLM-as-a-judge patterns, benchmark response quality, analyze cost/latency trade-offs, and build A/B testing frameworks.

## Key Topics

- Defining evaluation metrics for agentic systems
- LLM-as-a-judge automated evaluation
- Response quality benchmarking
- Latency, cost, and throughput analysis
- A/B testing and comparison frameworks
