# Session 3 — Student Notebook

## Model Evaluation and Comparison

### Setup Instructions

1. Ensure you have completed Session 1 setup (Python, packages, API key)
2. Open the notebook:
   ```bash
   jupyter notebook Session3_Student_Model_Evaluation.ipynb
   ```

### What's Inside

- **5 Demos** — Follow along to see evaluation techniques in action.
- **4 Hands-On Tasks** — Complete the TODO sections to build your own evaluation framework.

### Task Summary

| Task | Topic | Difficulty |
|---|---|---|
| Task 1 | Create a custom evaluation rubric | Beginner |
| Task 2 | Compare multiple models on standardized tasks | Intermediate |
| Task 3 | Build an automated evaluation pipeline | Intermediate |
| Task 4 | Analyze and visualize evaluation results | Advanced |

### Tips

- Think about what "good" means for an agentic response before defining metrics
- Use consistent test prompts across models for fair comparison
- Task 3 brings together scoring + comparison into one pipeline
- For Task 4, matplotlib charts will help communicate your findings clearly
