# Session 2 — Student Notebook

## Prompt Engineering for Agentic Behaviors

### Setup Instructions

1. Ensure you have completed Session 1 setup (Python, packages, API key)
2. Open the notebook:
   ```bash
   jupyter notebook Session2_Student_Prompt_Engineering.ipynb
   ```

### What's Inside

- **5 Demos** — Follow along with the instructor to see prompting techniques in action.
- **4 Hands-On Tasks** — Complete the TODO sections to build your own agentic prompts.

### Task Summary

| Task | Topic | Difficulty |
|---|---|---|
| Task 1 | Design system prompts for a research agent | Beginner |
| Task 2 | Implement ReAct-style prompting | Intermediate |
| Task 3 | Create a reusable prompt template library | Intermediate |
| Task 4 | Build a multi-step reasoning agent | Advanced |

### Tips

- Pay close attention to how small prompt changes lead to very different outputs
- Experiment with different persona descriptions in Task 1
- ReAct (Task 2) follows a Thought → Action → Observation loop — keep this pattern
- For Task 4, break the problem into steps before writing the prompt
