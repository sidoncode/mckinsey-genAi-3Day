### Key Teaching Points

1. Integration is where real value emerges — individual pieces combine into something greater
2. Evaluation-driven refinement is a continuous loop, not a one-time step
3. Production agents MUST handle errors gracefully — show failure modes
4. Documentation is part of engineering — encourage clear system descriptions

### Common Student Questions

- "How do I know when my agent is good enough?" — Define acceptance criteria upfront using evaluation metrics
- "Should I optimize prompts or switch models?" — Try prompt optimization first; it's cheaper and faster
- "What about deploying to production?" — Covered in later days; today focuses on the prototype
