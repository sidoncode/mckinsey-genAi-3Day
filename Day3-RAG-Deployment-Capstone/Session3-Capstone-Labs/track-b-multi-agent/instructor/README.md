## McKinsey Engagement Team (Multi-Agent Capstone)

### Teaching Notes

**Duration:** 1 hour 45 minutes (1:30 – 3:15)

### Suggested Timing

| Segment | Duration | Content |
|---|---|---|
| Intro & Setup | 5 min | Project overview — simulating a McKinsey engagement team |
| Milestone 1 | 20 min | EngagementManagerAgent supervisor & workstream decomposition |
| Milestone 2 | 25 min | Specialist consulting agents (Strategy, Financial, Operations, Industry) |
| Milestone 3 | 25 min | LangGraph engagement orchestration |
| Milestone 4 | 25 min | Partner Review quality loop |

| Milestone 6 | 15 min | Engagement analytics and reporting dashboard |

### What's Inside

- **Complete capstone solution** with detailed explanations at each milestone
- Ready to run end-to-end as a McKinsey Engagement Team reference

### Key Teaching Points

1. This capstone integrates Day 1 (LLM foundations), Day 2 (LangChain/LangGraph/multi-agent), and Day 3 (deployment patterns) — all modeling McKinsey engagement teams
2. Encourage students to think about failure modes — what happens when a specialist agent produces low-quality output?
3. The Partner Review quality loop is the most advanced part — it models the iterative refinement process of real consulting deliverables
4. Students should prepare a 2-minute demo for Session 4 cross-track presentations
