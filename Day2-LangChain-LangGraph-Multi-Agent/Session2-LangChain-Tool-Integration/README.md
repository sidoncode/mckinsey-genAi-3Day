# Session 2: LangChain Development & Tool Integration

**Time:** 11:00 – 12:45 (1 hour 45 minutes)

## Overview

This session introduces LangChain, the most widely adopted framework for building LLM-powered applications. Participants will learn its core abstractions — models, prompts, chains, and tools — and use LangChain Expression Language (LCEL) to compose modular, reusable pipelines that integrate with external tools and data sources.

## Key Topics

- LangChain architecture: ChatModels, PromptTemplates, OutputParsers
- LangChain Expression Language (LCEL) and the Runnable interface
- Custom tool creation and integration
- Document loaders and text splitters
- Retrieval-Augmented Generation (RAG) basics
